# CONTEXT — kimi-toolchain

> `https://github.com/brendadeeznuts1111/kimi-toolchain`

## Domain

Bun-native extension layer for Kimi Code and project governance. Provides diagnostics, MCP bridge provisioning, scaffolding, causal tracing, capability probes, signed contract validation, and safe self-healing plans synced into `~/.kimi-code/`. Does not replace the official Moonshot `kimi` agent (see `UNIFIED.md`).

## Architecture

### Repo Structure

```
kimi-toolchain/
  src/bin/            # CLI entry points
  src/lib/            # Shared library modules
  src/install-hooks/  # postinstall.ts sets up ~/.kimi-code/
  src/kimi-hooks/     # Kimi Code lifecycle hooks
```

Live runtime under `~/.kimi-code/` is managed by `postinstall.ts`; sync with `bun run sync`.
